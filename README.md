# Flight Control System V&V Portfolio Project
## Project Objective
This project demonstrates system-level Verification & Validation activities for a simplified aircraft flight control function. The system is based on an Airbus A320-style commercial aircraft concept, but the model is intentionally simplified for portfolio and learning purposes.
## Focus Areas
-System requirements definition
-System boundary definition
-Input and output analysis
-Functional safety considerations
-Verification planning
-Test case design
-Requirements traceability
## System Under Test
The System Under Test is a simplified Altitude Hold Control Function. The function receives aircraft state data and pilot/autopilot commands, then generates pitch and thrust commands to help maintain a target altitude.
